# command-line-game
A simple command line game in Java
![Game](images/game.png)


# Build the game
```bash
./gradlew clean build
```

# Build the game with code coverage report
```bash
./gradlew clean build jacocoTestReport

```

# Run the game
```bash
java -jar build/libs/clgame-1.0-SNAPSHOT.jar 

```
